# Event Assistant AI Agent

An AI-powered Event Planning Assistant that generates:
- Event timelines
- Vendor suggestions
- Invitation cards
- Memory-based refinement

Built using Python, tools, a custom agent architecture, and local memory.

## Features
- Timeline generation
- Vendor search using CSV
- Invite card generator
- Memory-based refinement

## How to Run
1. Install requirements:

2. Open notebook.ipynb in VS Code  
3. Run each cell  
4. Modify and test with your event details  

## File Structure
- agents.py → Main agent logic  
- tools.py → Tools: timeline, vendor search, invite creation  
- memory.py → Stores last plan  
- sample_vendors.csv → Vendor dataset  
- notebook.ipynb → Demo notebook  
- thumbnail.png → For Kaggle submission
